# 統計検定2級 学習プロジェクト

このリポジトリは、統計検定2級の合格を目指して、Pythonを用いた実践的な学習を行うためのプロジェクトです。
単なる公式の暗記ではなく、コードを動かし、データを可視化することで「生きた統計学」を身につけることを目的としています。

## 特徴

*   **インタラクティブな学習**: `marimo` を使用したリアクティブなノートブックで、パラメータを動かしながら統計的性質を実験できます。
*   **実践的なコード**: `pandas`, `numpy`, `matplotlib`, `seaborn` など、実務で使われるライブラリを使用しています。
*   **体系的なカリキュラム**: 統計検定2級の出題範囲を網羅するように構成されています。

## 実行方法

このプロジェクトは [marimo](https://marimo.io/) を使用しています。

### ローカル環境での実行

必要なライブラリをインストール後、以下のコマンドでノートブックを起動してください。

```bash
pip install marimo pandas numpy matplotlib seaborn
marimo edit 1.py
```

### Dockerでの実行

Docker環境も用意しています。

```bash
docker-compose up
```

起動後、ブラウザで `http://localhost:2718` にアクセスしてください。
